Repositorio de práctica para aprender GitHub y documentar estudios, ensayos y análisis profesionales.

Objetivo del repositorio

Este repositorio tiene como objetivo aprender a utilizar GitHub como plataforma de publicación, organización y versionamiento de documentos técnicos.

El foco principal está en contenidos relacionados con finanzas, contabilidad, costos, control de gestión, tributación y análisis financiero.
