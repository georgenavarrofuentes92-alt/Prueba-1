Prueba-1

Este repositorio fue creado como práctica inicial para aprender a utilizar GitHub como plataforma de publicación de ensayos, estudios y análisis profesionales.

Objetivo

El objetivo de este repositorio es practicar la creación, edición y publicación de documentos en GitHub, usando una estructura simple y ordenada.

Temas de interés

Este espacio podrá ser utilizado para desarrollar contenidos relacionados con:

* Finanzas
* Contabilidad
* Costos
* Control de gestión
* Tributación
* Evaluación de empresas
* Análisis financiero
* Programación aplicada a finanzas

Estructura inicial

La estructura sugerida para este repositorio es:

* ensayos/: textos de análisis, opinión o reflexión.
* estudios/: documentos más técnicos o con metodología.
* datos/: bases públicas o archivos de respaldo.
* graficos/: imágenes, tablas o visualizaciones.

Primer ejercicio

El primer ejercicio será crear un archivo llamado:

ensayos/mi-primer-ensayo.md

En ese archivo se desarrollará un primer ensayo de prueba usando formato Markdown.
